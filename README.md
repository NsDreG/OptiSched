# OptiSched
Infomatrix Asia 2026 project

This project is developed for InfoMatrix Asia 2026 - AI Programming Section.

Problem is that
Students spend a lot of time and mental effort creating and updating study schedules.
Manual planning is inefficient and stressful, especially when unexpected events occur.

Solution is to create
A web-based AI system that:
  Generates an optimized weekly study timetable
  Accepts natural language updates via chatbot
  Re-optimizes the schedule automatically
  Explains its decisions

Features
  Timetable input
  AI-based priority scheduling
  Conflict-free optimization
  Natural language chatbot
  Weekly calendar visualization
  PDF export
